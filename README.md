# Figma Token Migrator

Plugin para migrar Color Styles legados para Variables (Design Tokens) no Figma.

## Objetivo

- Escanear Color Styles
- Inferir tokens semânticos
- Criar Variables
- Migrar componentes
- Exibir fila de revisão para casos ambíguos

## MVP

- Auditoria de Color Styles
- Detecção de uso em Text, Fill e Stroke
- Geração de tokens text/* background/* border/*
- Relatório de confiança
